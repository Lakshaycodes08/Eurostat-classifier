Install

Unix / macOS / Linux / WSL

One command (latest release):

curl -fsSL https://swytchcode.gitlab.io/cli/install.sh | sh

Pinned version

VERSION=v0.1.5 curl -fsSL https://swytchcode.gitlab.io/cli/install.sh | sh

Custom install directory

INSTALL_DIR=$HOME/bin curl -fsSL https://swytchcode.gitlab.io/cli/install.sh | sh

Default: /usr/local/bin if writable, else ~/.local/bin.

Windows (PowerShell)

irm https://swytchcode.gitlab.io/cli/install.ps1 | iex

Or download the .zip from releases and extract.

What the script does

So you can trust it:

  • Detects your OS and architecture
  • Downloads the matching binary from GitLab Releases
  • Downloads checksums.txt and verifies SHA256
  • Extracts and installs the binary
  • Does not execute anything automatically – no hidden steps

No login, no API token, no saving a file first. One command.
