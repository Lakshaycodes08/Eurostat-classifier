Swytchcode CLI

Latest version: __LATEST_VERSION__

Execution authority for agentic API integrations.
The core execution kernel that enforces policy, determinism, and safety between AI agents and real systems.

Swytchcode sits between AI agents and production APIs, ensuring execution is explicit, controlled, and never implicit or prompt-driven.

Install

curl -fsSL https://cli.swytchcode.com/install.sh | sh

Supports macOS and Linux on x64 and arm64.

irm https://cli.swytchcode.com/install.ps1 | iex

Run in PowerShell. Supports x64 and arm64.

npm install -g swytchcode

Requires Node.js 14+. Works on macOS, Linux, and Windows.

The shell and PowerShell scripts download the correct binary for your OS and architecture, verify checksums, and install locally without requiring Node.js.
