Frontend / agent example

Call swytchcode exec from a frontend or agent via JSON stdin:

echo '{"tool":"api.cluster.create","args":{...}}' | swytchcode exec

Or run the MCP server and connect your app to it: swytchcode mcp serve. The server exposes swytchcode_exec and other tools over stdio or HTTP.

For full spec (request/response schema, exit codes), see Wiki → CLI Reference.
