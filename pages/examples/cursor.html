Cursor integration

Use Swytchcode from Cursor so the agent calls swytchcode exec instead of running tools directly.

  1. Install Swytchcode: Install
  2. In your project: swytchcode init --editor=cursor
  3. Add tools to tooling.json with swytchcode add <canonical_id>
  4. Cursor uses the rule in .cursor/rules/swytchcode.mdc to delegate execution to swytchcode exec.

For full integration details, see Wiki → Integrations → Cursor.
